C++ Primer (5th Edition)

As a C++ beginner developer, I find this book very interesting. It teaches from basic aspects of this language to more advanced topics. Also it introduces the c++11 standard which introduces modern features like data structures, smart pointers and algorithms that makes the development of software easier and more secure.

Clean Code: A Handbook of Agile Software Craftsmanship

This book explains how to write clean code with a series of tips that every programmer should know in order to write code that is easy to understand. The best parts of it are the chapters that focus on classes, functions and code smells.

Game Programming Patterns

Writting video games can be hard and the code can become messy and unmaintainable. Knowing about game programming patterns is essential in the game dev industry. It has helped to me to structure some parts of the code, making it better and easy to modify.

Professional CMake

A good guide for begginers with CMake, wrote by one of the creators of this software.

SFML Game Development

This book is a good start for an introduction to game development with the SFML library. It explains how to make a basic engine for a videogame, the basics of game programming and is easy to understand.
