Interesting books

This section is about the books that I have read and I find useful in order to improve as a developer and as a computer engineer.

  • C++ Primer (5th Edition)
  • Clean Code: A Handbook of Agile Software Craftsmanship
  • SFML Game Development

C++ Primer (5th Edition)

As a C++ beginner developer, I find this book very interesting. It teaches from basic aspects of this language to more advanced topics. Also it introduces the c++11 standard which introduces modern features like data structures and algorithms that helps to develop in an easier way.

Clean Code: A Handbook of Agile Software Craftsmanship

This book explains how to write clean code with a serie of tips that I find useful. The best parts of it are the chapters that focus on classes, functions and code smells. One of the aspects that I don't like about the book is that it is Java focused and there are some parts that aren't interesting.

SFML Game Development

This book is a good start for an introduction to game development with the SFML library. It explains how to make a basic engine for a videogame and is easy to understand.
